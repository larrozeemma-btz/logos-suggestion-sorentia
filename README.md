# Sorentia — Logo Exploration

Brand identity exploration for Sorentia, a platform built around the idea that everything can be rented.

## Concept

This round explores six logo directions based on a core visual system:
an elongated oval shape paired with a punctuated point.

Each direction reflects a different positioning:
premium, editorial, modern, accessible.

## Directions

01 — Integrated mark  
02 — Elevated wordmark  
03 — Balanced system  
04 — Anthropomorphic approach  
05 — Monogram  
06 — Lifestyle / app direction  

## Next step

Selection of 1–2 directions for refinement.
